# LA Emu AI ROI Calculator

A web application for calculating AI marketing ROI.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Lucide Icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Build

```bash
npm run build
```
